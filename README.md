# 🚚 Supply Chain Analytics Dashboard

> An end-to-end supply chain analytics solution on 110K+ global orders — featuring late delivery prediction (98.33% accuracy), fraud detection, demand forecasting, and profitability analysis — deployed as an interactive Streamlit dashboard.

## 🚀 Live Demo
👉 [Click here to view the live dashboard[]()](https://fwptza6nyqnpn4xhicgwam.streamlit.app/)

## 📌 Project Overview
- Analyzed **110,974 orders** across global supply chain (2015–2018)
- Built **ML models** for late delivery prediction with **98.33% accuracy**
- Detected **suspicious/fraudulent orders** using Isolation Forest
- Built **demand forecasting** model using Gradient Boosting Regressor
- Deployed **5-page interactive Streamlit dashboard**

## 📊 ML Model Comparison
| Model | Accuracy | Notes |
|-------|----------|-------|
| Logistic Regression | 98.28% | Fast, interpretable |
| **Random Forest** | **98.33% 🏆** | Best overall |
| Gradient Boosting | 98.28% | Close competitor |

## ✨ Dashboard Features
- **🏠 Executive Overview** — KPI cards, sales by market, delivery status, monthly trends
- **🚚 Delivery Performance** — Late delivery by shipping mode & region, delay distribution
- **💰 Profitability Analysis** — Top categories, customer segments, department performance
- **🔍 Fraud Detection** — Suspicious order patterns, fraud rate, category-wise analysis
- **🤖 Late Delivery Predictor** — Real-time ML prediction with confidence score
